KeyLimePy

Automate your CraftBukkit server painlessly

Download as .zip Download as .tar.gz View on GitHub

Instructions

  • Make sure that the CraftBukkit .jar file is located in a 'craftbukkit' folder in your home directory on your server computer
  • Rename the CraftBukkit .jar file to craftbukkit.jar.
  • Download all the keylime.py files from the GitHub repository.
  • Set up a GMail account for your server.
  • Set up a Google Voice number for your server.
  • In a command line, navigate to the folder where you're storing the keylime.py files.
  • Create a file named 'emails.txt' with the emails of your server's members on separate lines.
  • Type python keylime.py in your command line.
  • Type in the information that the program asks you for. You can find your external IP address at www.whatismyipaddress.com.
  • Enjoy a fully automated server!

Features

keylime.py knows when your server is not running, and automatically emails server members about the server's downtime. keylime.py also knows when/if your server's IP address has changed, and emails server members a notice about that.

For remote server management, keylime.py allows you to email commands to be run on your server. Just send an email to your server's GMail account with the authentication code as the subject, and the server command as the message. You'll receive a confirmation email with the new authentication code.

You can also text commands to your server! Send a text to your server's Google Voice number with the authentication code, then a space, then the server command you want to run. You'll receive a confirmation text, and a text with the new authentication code.
